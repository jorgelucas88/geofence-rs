# Geofence API

## Description
Provides an API to calculate if a pair of coordinates is within a geofence polygon

## Node version
Node v14

## NestJS
NestJS v8.2.3

## Installation
- set to node 14
- clone repo branch dev
- npm install

To run
- Using docker: docker-compose up dev
- Running locally: npm run start
